This is an open-source training program for programmers and software engineers.

Philosophy

Programming really isn't about any one programming language, so neither is this course. Programming is about deconstructing problems and thinking abstractly about how to solve them.

This course starts with the foundations of software and builds, one idea at a time, to illustrate the many abstractions and layers which exist below a program, and how to use layers and abstractions to build solid programs.
